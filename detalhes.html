📋 Sobre o Evento

Workshop prático de desenvolvimento web focado em tecnologias front-end. Os participantes aprenderão a criar sites responsivos e modernos utilizando HTML5, CSS3 e JavaScript, com exercícios práticos e projeto final.

Ideal para estudantes, profissionais em transição de carreira e entusiastas que desejam iniciar ou aprimorar seus conhecimentos em desenvolvimento web.

🎯 Objetivos

  • Compreender a estrutura semântica de páginas web com HTML5
  • Criar layouts responsivos e acessíveis utilizando CSS3
  • Implementar interatividade em páginas web com JavaScript moderno
  • Desenvolver um projeto prático do zero para o portfólio
  • Conhecer boas práticas de desenvolvimento e versionamento com Git
